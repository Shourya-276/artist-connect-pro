# Live101 - Artist Booking Platform

## Project Overview
Live101 is India's premier marketplace for booking live artists for your events. This platform connects event planners with a diverse range of talent, including singers, DJs, bands, comedians, and more.

## Key Features
- **Browse by Category**: Easily find artists based on their specialty.
- **Search & Filter**: Find artists by name, category, or city.
- **Artist Profiles**: View detailed profiles, ratings, and reviews.
- **Booking System**: Streamined process for booking talent for your events.
- **Artist & Client Dashboards**: Dedicated portals for managing profiles and event bookings.

## Technology Stack
- **Vite**: Build tool and development server.
- **React**: UI library.
- **TypeScript**: Static typing.
- **Tailwind CSS**: Styling.
- **Framer Motion**: Animations.
- **Lucide React**: Icon library.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Local Development
The development server will move to `http://localhost:8080` by default.

## Deployment
Build the project for production:
```sh
npm run build
```
The output will be in the `dist/` directory.
